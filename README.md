## 概要

イベントの情報を登録し、一覧表示する簡単な React アプリケーションです。  
各オペレーション（イベント作成、削除等）のログも一覧表示します。  

## 使用API・パッケージ・他

- Hooks
  - useState
  - useEffect
  - useReducer
  - useContext

- Redux
  - combineReducers

- Sass(SCSS) + BEM
