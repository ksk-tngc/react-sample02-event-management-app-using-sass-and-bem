## 概要

イベント情報を作成し、一覧表示する簡単な React アプリケーションです。  
各オペレーション（イベント作成、削除等）のログも一覧表示します。  

以下の要素を含みます：

* Sass (SCSS) + BEM
* localStorage
* Hooks
  - useState
  - useEffect
  - useReducer
  - useContext
* Redux
  - combineReducers
